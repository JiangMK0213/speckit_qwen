# 类: Boundary

**命名空间:** THING.EXTEND

**描述:** 构造函数

**参数:**

| 名称 | 类型 | 默认值 | 可选 | 描述 |
|------|------|--------|-------------|--------|
| param | `object` | - | 否 | 参数对象 |
| param.height | `Number, Array.<Number>` | - | 否 | 围栏高度 支持数组 长度需要和points或者selfPoints一致 |
| param.isAcceptLight | `Boolean` | - | 否 | 是否接受光照 |
| param.points | `Array.<number>` | - | 否 | 点集合 |
| param.selfPoints | `Array.<number>` | - | 否 | 自身坐标系下的点集合 |
| param.style | `Style` | - | 否 | 围栏样式 |

## 实例属性

### points

**类型:** `member`

**数据类型:** `Array.<Number>`

**描述:** 获取/设置点集合

---

### selfPoints

**类型:** `member`

**数据类型:** `Array.<Number>`

**描述:** 获取/设置自身坐标系下的点集合

---

### height

**类型:** `member`

**数据类型:** `Number, Array.<Number>`

**描述:** 获取/设置高度 支持数组

---

### isAcceptLight

**类型:** `member`

**数据类型:** `Boolean`

**描述:** 获取/设置是否接受光照

---
