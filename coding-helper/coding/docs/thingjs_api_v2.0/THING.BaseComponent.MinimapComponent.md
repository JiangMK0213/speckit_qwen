# 类: MinimapComponent

**命名空间:** THING.BaseComponent

**描述:** 小地图组件

**参数:**

| 名称 | 类型 | 默认值 | 可选 | 描述 |
|------|------|--------|-------------|--------|
| param | `object` | - | 否 | 初始参数 |

## 实例属性

### size

**类型:** `member`

**数据类型:** `Array`

**描述:** 设置/获取小地图的尺寸

#### 示例

```javascript
minimap.size = [300, 300]
```

---

### layout

**类型:** `member`

**数据类型:** `Array`

**描述:** 设置/获取小地图的布局

#### 示例

```javascript
minimap.layout = ["Left", "Bottom"]
```

---

### position

**类型:** `member`

**数据类型:** `Array`

**描述:** 设置/获取小地图的位置

#### 示例

```javascript
minimap.position = [1,1]
```

---

### enable

**类型:** `member`

**数据类型:** `Array`

**描述:** 设置/获取小地图是否启用

#### 示例

```javascript
minimap.enable = true
```
