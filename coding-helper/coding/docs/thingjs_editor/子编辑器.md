编辑器通过 `openEditor` 方法打开子编辑器。每个子编辑器拥有独立的 `editor` 对象，接口与主编辑器完全相同，支持插件依赖、事件监听以及跨窗口通信。

## 基础使用

### 打开和关闭子编辑器

```javascript
// 打开子编辑器，并返回其 ID
const editorId = editor.openEditor(
  '/path/sub-editor/index.js',  // 子编辑器入口文件地址
  '我的子编辑器'  // 窗口标题（可选）
)

// 关闭子编辑器
editor.closeEditor(editorId)
```

### 子编辑器入口文件示例

`index.js` 作为子编辑器的入口文件，创建并挂载 Vue 根组件。

```javascript
import { createApp } from 'vue'
import MySubEditor from './MySubEditor.vue'

// 创建 Vue 实例并传递参数
const app = createApp(MySubEditor, { someData: 'test-data' })
const div = document.createElement('div')
app.mount(div)
```

### 子编辑器根组件示例

在子编辑器的根组件中，可以获取传递的参数，并访问 `editor` 对象。

```javascript
import { onMounted } from 'vue'

// 获取传递的参数
const props = defineProps(['someData'])
console.log(props.someData)  // 输出: test-data

// 组件挂载完成时执行
onMounted(() => {
  console.log(editor)  // 子编辑器的 editor 对象
})
```

## 事件监听

在关闭子编辑器前，可以注册 `close` 事件处理逻辑，以确保数据保存完毕再退出。

```javascript
// 定义关闭前的异步处理逻辑
function handleClose() {
  return new Promise((resolve) => {
    setTimeout(() => resolve(), 3000)  // 模拟保存操作
    // setTimeout(() => resolve(false), 3000) // 返回 false 可以阻止窗口关闭
  })
}

onMounted(() => {
  editor.on('close', handleClose)  // 注册关闭事件
})

onUnmounted(() => {
  editor.off('close', handleClose)  // 移除关闭事件
})
```

## 插件依赖

可以在打开子编辑器时传入依赖插件，并在子编辑器中访问这些插件。

### 示例：指定依赖插件

```javascript
editor.openEditor(
  '/path/prefab-editor/index.js', 
  '预制件编辑器', 
  ['plugin1', 'plugin2']  // 子编辑器依赖的插件
)

// 在子编辑器中获取插件并调用方法
const plugin1 = editor.getPlugin('plugin1')
plugin1.someMethod()
```

### 设置依赖插件的启动参数

```javascript
editor.openEditor(
  '/path/prefab-editor/index.js', 
  '预制件编辑器', 
  [
    {
      name: 'plugin1',
      params: { data: 123 }
    }
  ]
)

// 在插件入口文件中获取参数
export default {
  onBeforeInstall(params) {
    console.log(params)  // 输出: { data: 123 }
  },
  onInstall(params) {
    console.log(params)  // 输出: { data: 123 }
  }
}
```

## 跨窗口通信

支持主编辑器与子编辑器之间的双向通信。

### 主编辑器：监听子编辑器消息

```javascript
// 打开子编辑器并获取其 ID
const editorId = editor.openEditor('/path/sub-editor/index.js')

// 消息处理函数，只处理来自该子编辑器的消息
function handleMessage(message, id) {
  if (id !== editorId) return  // 过滤其他子编辑器的消息
  console.log(message)  // 输出: data-from-subeditor
}

// 注册消息监听
editor.addMessageListener(handleMessage)

// 移除消息监听，防止插件卸载时内存泄漏
editor.removeMessageListener(handleMessage)
```

### 子编辑器：发送消息给主编辑器

```javascript
// 子编辑器发送消息给主编辑器
editor.postMessage('data-from-subeditor')
```
