# Code Performance Checker

## 功能描述

此skill用于检查和分析代码的性能，帮助开发者识别性能瓶颈并提供优化建议。

## 核心功能

### 1. 执行时间分析
- 测量代码执行时间
- 识别执行时间较长的代码片段
- 分析函数调用时间分布

### 2. 内存使用分析
- 检查代码的内存使用情况
- 识别可能的内存泄漏
- 分析内存分配和释放模式

### 3. 代码复杂度分析
- 分析代码的时间复杂度
- 分析代码的空间复杂度
- 识别高复杂度的代码片段

### 4. 优化建议
- 基于分析结果提供具体的性能优化建议
- 推荐算法和数据结构的改进
- 提供代码重构建议

## 使用方法

当用户需要：
- 分析代码的执行性能
- 检查代码的内存使用情况
- 优化代码的时间复杂度
- 获得性能改进建议时

调用此skill，它将分析指定的代码并提供详细的性能分析报告和优化建议。
