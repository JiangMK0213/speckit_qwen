`Storage` 模块为开发者提供了文件和路径操作接口，底层基于 Node.js 的 `fs/promises`。所有方法都需要与主进程通信，因此均为异步操作。建议开发者在调用时配合 `try-catch` 进行异常处理。

## 目录路径获取

```javascript
/**
 * 获取路径
 * @param {string} name - 路径名称，可选值如下：
 *   - 'project': 项目目录路径
 *   - 'app': 编辑器所在目录路径
 *   - 'userData': 用户数据目录路径
 *   - 'temp': 临时目录路径
 * @returns {Promise<string>} 返回对应的路径字符串
 */
const projectPath = await editor.storage.getPath('project')
```

## 文件操作

### 写文件

```javascript
/**
 * 写入文件
 * @param {string} path - 文件路径
 * @param {string | Buffer} data - 要写入的内容
 * @returns {Promise<void>}
 */
const content = 'Hello World!'
await editor.storage.writeFile(path, content)
```

### 写图片

```javascript
/**
 * 写入图片
 * @param {string} path - 图片保存路径
 * @param {string | ArrayBuffer} imageData - 图片数据（Base64 或 ArrayBuffer）
 * @returns {Promise<void>}
 */
await editor.storage.writeImage(path, imageData)
```

### 读文件

```javascript
/**
 * 读取文件
 * @param {string} path - 文件路径
 * @returns {Promise<string>} 文件内容字符串
 */
const content = await editor.storage.readFile(path)
```

### 读图片

```javascript
/**
 * 读取图片
 * @param {string} imgPath - 图片路径
 * @returns {Promise<string>} 图片的 Base64 字符串
 */
const base64Image = await editor.storage.readImage(imgPath)
```

### 删除文件

```javascript
/**
 * 删除文件
 * @param {string} path - 文件路径
 * @returns {Promise<void>}
 */
await editor.storage.deleteFile(path)
```

### 获取文件大小

```javascript
/**
 * 获取文件大小（字节）
 * @param {string} filePath - 文件路径
 * @returns {Promise<number>} 文件大小（字节数）
 */
const size = await editor.storage.getFileSize(filePath)
console.log(`File size: ${size} bytes`)
```

## 文件夹操作

### 创建文件夹

```javascript
/**
 * 创建文件夹
 * @param {string} path - 文件夹路径
 * @returns {Promise<void>}
 */
await editor.storage.createFolder(path)
```

### 读取文件夹

```javascript
/**
 * 读取文件夹内容
 * @param {string} path - 文件夹路径
 * @returns {Promise<string[]>} 文件名数组

const files = await editor.storage.readFolder(path)
```

### 删除文件夹

```javascript
/**
 * 删除文件夹
 * @param {string} path - 文件夹路径
 * @returns {Promise<void>}
 */
await editor.storage.deleteFolder(path)
```

## 路径与文件类型判断

### 判断路径是否存在

```javascript
/**
 * 判断路径是否存在
 * @param {string} path - 路径
 * @returns {Promise<boolean>} 路径是否存在
 */
const exists = await editor.storage.existsPath(path)
```

### 判断路径是否是文件夹

```javascript
/**
 * 判断路径是否是文件夹
 * @param {string} path - 路径
 * @returns {Promise<boolean>} 是否为文件夹
 */
const isFolder = await editor.storage.isFolder(path)
```

### 判断路径是否是文件

```javascript
/**
 * 判断路径是否是文件
 * @param {string} path - 路径
 * @returns {Promise<boolean>} 是否为文件
 */
const isFile = await editor.storage.isFile(path)
```

### 拼接路径

```javascript
/**
 * 拼接路径
 * @param {...string} segments - 路径片段
 * @returns {Promise<string>} 拼接后的路径
 */
const fullPath = await editor.storage.joinPath('folder', 'subfolder', 'file.txt')
```

## 文件拷贝、移动与重命名

### 拷贝文件或文件夹

```javascript
/**
 * 拷贝文件或文件夹
 * @param {string} source - 源路径
 * @param {string} destination - 目标路径
 * @returns {Promise<void>}
 */
await editor.storage.copy(source, destination)
```

### 移动文件或文件夹

```javascript
/**
 * 移动文件或文件夹
 * @param {string} source - 源路径
 * @param {string} destination - 目标路径
 * @returns {Promise<void>}
 */
await editor.storage.move(source, destination)
```

### 重命名文件或文件夹

```javascript
/**
 * 重命名文件或文件夹
 * @param {string} oldPath - 旧路径
 * @param {string} newPath - 新路径
 * @returns {Promise<void>}
 */
await editor.storage.rename(oldPath, newPath)
```

## 压缩与解压缩

### 压缩

```javascript
/**
 * 压缩文件或文件夹
 * @param {string | string[]} sourcePath - 源路径或路径数组
 * @param {string} zipPath - 压缩包保存路径
 * @param {boolean} [includeFolder=true] - 是否包含文件夹
 * @returns {Promise<boolean>} 是否成功压缩
 */
await editor.storage.zip(sourcePath, zipPath, true)
```

### 解压缩

```javascript
/**
 * 解压缩文件
 * @param {string} zipPath - 压缩包路径
 * @param {string} extractPath - 解压路径
 * @returns {Promise<boolean>} 是否成功解压
 */
await editor.storage.unzip(zipPath, extractPath)
```
