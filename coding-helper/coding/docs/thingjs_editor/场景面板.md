场景面板组件用于管理和展示场景，包含场景的添加、删除、显隐和层级调整等。

![](https://pan.uino.cn/files/image/2024/9/c69eecda680247df8c354b54fdada92d.png)

## 使用场景面板

```javascript
// 使用场景面板
const scene = editor.usePanel('Scene')
```

## 方法

### addPage

`addPage(option: Object)`

添加页面，返回 page。  
**参数**

|参数名|类型|描述|
|---|---|---|
|option|`object`|页面信息|

**option**

|参数名|类型|描述|
|---|---|---|
|id|`string`|页面 id|
|name|`string`|页面名称|
|data|`Object`|自定义数据|
|bounds|`Object`|页面大小和边距设置|

**用法**

```javascript
const page = scene.addPage({
  id: 'page-3d',
  name: '3D',
  data: { // 可选，自定义数据
    ...
  },
  bounds: { // 可选，默认值为
    width: 100,
    height: 100,
    left: 0,
    top: 0,
    widthUnit: '%',
    heightUnit: '%',
    leftUnit: '%',
    topUnit: '%'
  }
})
```

### removePage

`removePage(id：String)`

删除页面。

**参数**

|参数名|类型|描述|
|---|---|---|
|id|`string`|页面 id|

**用法**

```javascript
scene.removePage('page-chart')
```

### copyPage

`copyPage(id：String, option: Object)`

复制页面，返回复制的 page。

**参数**

|参数名|类型|描述|
|---|---|---|
|id|`string`|页面 id|
|option|`object`|页面信息|

**用法**

```javascript
scene.copyPage('page-chart', options) // 复制的 page，缺省 id 为原有 id + '-copy'，name 为原有 name + '-副本'；也可在 options 中指定 id 和 name
```

### renamePage

`renamePage(id：String, name: String)`

重命名页面。

**参数**

|参数名|类型|描述|
|---|---|---|
|id|`string`|页面 id|
|name|`string`|新名称|

**用法**

```javascript
scene.renamePage('page-chart', '我的大屏')
```

### getPage

`getPage(id：String】)`

获取页面。

**参数**

|参数名|类型|描述|
|---|---|---|
|id|`string`|页面 id|

**用法**

```javascript
const page = scene.getPage('page-chart')
```

### 使用示例

```javascript
import { nextTick } from 'vue'

// 使用场景面板
const scene = editor.usePanel('Scene')

// 添加页面，返回 page
const page = scene.addPage({
  id: 'page-3d',
  name: '3D',
  data: { // 可选，自定义数据
    ...
  }
})

// 确保 DOM 已更新，也可用 setTimeout
await nextTick()

// 使用页面的 DOM 容器
const container = page.getContainer()
const app = new THING.App({
  container
})

// 删除页面
scene.removePage('page-chart')

// 复制页面，返回复制的 page
scene.copyPage('page-chart', options) // 复制的 page，缺省 id 为原有 id + '-copy'，name 为原有 name + '-副本'；也可在 options 中指定 id 和 name

// 重命名页面
scene.renamePage('page-chart', '我的大屏') // 参数为页面 id 和新的名称

// 获取页面
const page = scene.getPage('page-chart')

// 激活页面
page.setActive(true)

// 获取场景的 page 页信息
const pages = scene.pages

// 常显页面
page.setVisible(true)
```
