蓝图面板用于组织场景中的流程和逻辑。

![](https://pan.uino.cn/files/image/2024/9/9123d294dead9ce64c71397813261999.png)

## 使用蓝图面板

```javascript
// 使用蓝图面板
editor.usePanel('Blueprint')
```

## 蓝图对象

蓝图面板的方法通过 editor 上的蓝图对象提供。

```javascript
// 蓝图对象
const blueprint = editor.blueprint
```

## 属性

### currentPage

获取当前页。

**用法**

```javascript
// 获取当前页
blueprint.currentPage
```

### contextMenuData

获取右键菜单数据。

```javascript
// 获取右键菜单数据
blueprint.contextMenuData
```

## 方法

### setContainer

`setContainer(container：HTMLElement)`

设置 DOM 容器。

**参数**

|参数名|类型|描述|
|---|---|---|
|container|`HTMLElement`|容器元素|

**用法**

```javascript
/**
 * 设置 DOM 容器
 * @param container 容器元素
 */
blueprint.setContainer(container)
```

### registerNode

`registerNode(nodeClass: function | Array, options: Object):void`

注册节点。

**参数**

|参数名|类型|描述|
|---|---|---|
|nodeClass|`function或array`|节点的类|
|options|`object`|节点样式参数|

**用法**

```javascript
/**
 * 注册节点
 * @param nodeClass 节点的类
 * @param options 节点样式参数： 例：{ icon: 'base64 for my-bp-01.png',headerColor: '#ff00ff', bodyColor: 'rgba(11, 22, 33, 0.5)' }
 * @returns
 */
blueprint.registerNode(nodeClass, options)
```

### unregisterNode

`unregisterNode(name: String)`

取消注册节点。

**参数**

|参数名|类型|描述|
|---|---|---|
|name|`string`|节点名称|

**用法**

```javascript
/**
 * 取消注册节点
 * @param name 节点名称
 */
blueprint.unregisterNode(name)
```

### updateNode

`updateNode(nodeClass: Class | Array)`

更新注册的节点。

**参数**

|参数名|类型|描述|
|---|---|---|
|nodeClass|`class或array`|节点的类|

**用法**

```javascript
/**
 * 更新注册的节点
 * @param nodeClass 节点的类
 */
blueprint.updateNode(nodeClass)
```

### load

`load(json: BlueprintJSON)`

更新注册的节点。

**参数**

|参数名|类型|描述|
|---|---|---|
|json|`BlueprintJSON`|蓝图逻辑的 JSON|

**用法**

```javascript
/**
 * 加载蓝图 JSON
 * @param JSON 蓝图 JSON 文件
 */
blueprint.load(json)
```

### saveJSON

`saveJSON()`

保存蓝图 JSON（编辑时使用）。

**用法**

```javascript
blueprint.saveJSON()
```

### exportJSON

`exportJSON()`

导出蓝图 JSON（运行时使用）

**用法**

```javascript
blueprint.exportJSON()
```

### render

`render()`

渲染蓝图编辑器。

**用法**

```javascript
blueprint.render()
```

### stop

`stop()`

停止蓝图。

**用法**

```javascript
blueprint.stop()
```

### loadFolder

`loadFolder(contextMenuData：Object)`

加载右键菜单。

**参数**

|参数名|类型|描述|
|---|---|---|
|contextMenuData|`object`|右键菜单配置信息|

**contextMenuData**：

|参数名|类型|描述|
|---|---|---|
|foldername|`string`|文件夹名称|
|nodes|`array`|注册后节点的 config.name|
|children|`array`|子文件信息|

**用法**

```javascript
blueprint.loadFolder(contextMenuData)
```

### transformDomPos

`transformDomPos(position：Array)`

转换为蓝图文档坐标。

**参数**

|参数名|类型|描述|
|---|---|---|
|position|`array`|相对于蓝图面板的坐标|

**用法**

```javascript
blueprint.transformDomPos(position)
```

### addNode

`addNode(nodeName：String, inputObject: Object, mousePosition: Array)`

添加节点。

**参数**

|参数名|类型|描述|
|---|---|---|
|nodeName|`string`|节点名称|
|inputObject|`object`|要修改的节点输入的键值对|
|mousePosition|`array`|鼠标位置|

**用法**

```javascript
/**
 * 添加节点
 * @param nodeName 节点名称
 * @param inputObject 要修改的节点输入的键值对
 * @param mousePosition 鼠标位置
 * @returns 创建的节点
 */
blueprint.addNode(nodeName, inputObject, mousePosition)
```

### on

`on(eventType：String, listener: function)`

添加节点。

**参数**

|参数名|类型|描述|
|---|---|---|
|eventType|`string`|事件类型|
|listener|`function`|事件处理函数|

**用法**

```javascript
/**
 * 添加事件监听
 * @param eventType 事件类型 支持drop、run、stop、export、save以及[蓝图编辑器支持的事件](https://wiki.uino.com/book/612f5205dec53e3d4278d89a/619daa625dd1402d0d6eb5ca.html)
 * @param listener 事件处理函数
 */
blueprint.on(eventType, listener)
```
