编辑器为插件提供了一套简洁高效的国际化方案，实现语言切换和内容的动态翻译。

## 国际化文件配置

在插件的根目录下创建一个 `**i18n.json**` 文件，用于维护多语言内容。每个键代表默认语言的文本（如中文），其值为对应语言的翻译。

### 示例：i18n.json 文件

```bash
{
  "标题": {
    "en": "Title"  // 英文翻译
  },
  "内容": {
    "en": "Content"
  }
}
```

## 使用 i18n 方法实现国际化

通过编辑器提供的 `i18n` 方法，将 JSON 文件中的内容与当前语言关联，实现动态翻译。

### 示例：在 Vue 组件中使用国际化

#### 模板部分

```vue
<template>
  <div ref="template">
    <p>{{ t('内容') }}</p>  <!-- 在模板中调用 t 函数 -->
  </div>
</template>
```

#### 脚本部分

```javascript
<script setup>
import { ref, onMounted, onUnmounted } from 'vue'
import i18n from './i18n.json'

// 初始化 t 函数用于翻译
const t = editor.i18n(i18n)

// 定义模板引用
const template = ref()

onMounted(() => {
  // 添加面板，并使用 t 函数设置面板名称
  editor.addPanel({
    id: 'test',
    name: t('标题'),  // 脚本中调用 t 函数
    template
  })
})

onUnmounted(() => {
  // 移除面板
  editor.removePanel('test')
})
</script>
```

## 获取当前语言

可以通过访问 `editor.language` 获取编辑器当前的语言环境。例如：

```javascript
console.log(editor.language)  // 输出当前语言（如 'en' 或 'zh'）
```
