属性面板组件支持设置、获取和更新属性面板的配置。开发者还可以监听属性面板的变化，以便实现与其他面板的交互。

![](https://pan.uino.cn/files/image/2024/9/85411bd8e9f800079af035350b2e592b.png)

## 使用属性面板

```javascript
const inspector = editor.usePanel('Inspector')
```

## 属性

### enableAddComponent

是否启用添加自定义组件功能，默认为 `false`。启用后属性面板会显示“添加组件”按钮，组件列表会展示通过 `defineComponent` 方法预定义的组件。

## 方法

### getOptions

`getOptions(): InspectorPanelOptions`

获取当前属性面板的配置。

**返回值**：

* `InspectorPanelOptions`：属性面板的配置对象。

**用法**

```javascript
// 获取属性面板完整配置
inspector.getOptions()
```

### setOptions

`setOptions(options: InspectorPanelOptions): void`

设置属性面板的配置。

**参数**

|参数名|类型|描述|
|---|---|---|
|options|`InspectorPanelOptions`|要设置的属性面板配置对象|

属性的类型通过 type 字段来定义，具体配置项请参考属性面板 [配置规则](#配置规则)。

**用法**

```javascript
// 定义属性面板配置
inspector.setOptions({
  name: {
    name: '名称',
    type: 'text',
    value: '设备x'
  },
  entity: {
    name: '物体',
    type: 'group',
    children: {
      position: {
        name: '位置',
        type: 'vector3',
        value: [100, 100, 100]
      },
      angle: {
        name: '角度',
        type: 'vector3',
        value: [0, 0, 0]
      }
    }
  },
  tagsGroup: {
    name: '标签',
    type: 'group',
    children: {
      tags: {
        name: '标签',
        type: 'tag',
        value: ['标签1', '标签2']
      }
    }
  }
})
```

### updateOption

`updateOption(key: string, value: any): void`

手动修改属性面板中的一个属性值，并且会触发属性面板的 change 事件。

**参数**

|参数名|类型|描述|
|---|---|---|
|key|`string`|要修改的属性的键|
|value|`any`|要设置的新值|

**用法**

```javascript
// 手动修改属性值
inspector.updateOption('name', '设备xx')

// 手动修改嵌套的子项属性值
inspector.updateOption('tagsGroup.tags', ['标签3'])
```

### registerComponent

`registerComponent(name: string, config: CustomComponentOption, category?: string): void`

注册自定义组件。当属性面板的 `enableAddComponent` 属性为 `true` 时，自定义组件会显示在添加组件的列表中。组件被添加到属性面板后，会以 group 类型展示。

**参数**

|参数名|类型|描述|
|---|---|---|
|name|`string`|自定义组件名|
|config|`CustomComponentOption`|自定义组件配置|
|category|`string`|自定义组件分类，可选，不传则使用默认分类|

**用法**

```javascript
// 不设置分类
inspector.registerComponent('fog', {
  name: '雾设置',
  children: {
    color: {
      name: '颜色',
      type: 'color',
      value: '#FFFFFF'
    },
    density: {
      name: '强度',
      type: 'slider',
      min: 0,
      max: 0.1,
      step: 0.0005,
      value: 0.001,
      tooltip: '关于强度的描述信息'
    }
  }
})

// 第三个参数设置分类
inspector.registerComponent(
  'glow',
  {
    name: '发光通道',
    children: {
      strength: {
        name: '强度',
        type: 'slider',
        min: 0,
        max: 5,
        step: 0.01,
        value: 1.5
      },
      threshold: {
        name: '阈值',
        type: 'slider',
        min: 0,
        max: 1,
        step: 0.01,
        value: 0.01
      },
      radius: {
        name: '影响半径',
        type: 'slider',
        min: 0,
        max: 1,
        step: 0.01,
        value: 0.4
      }
    }
  },
  '发光设置'
)
```

### unregisterComponent

`unregisterComponent(name: string, category?: string): void`

注销自定义组件。

**参数**

|参数名|类型|描述|
|---|---|---|
|name|`string`|自定义组件名|
|category|`string`|自定义组件分类，可选|

**用法**

```javascript

```

### getComponent

`getComponent(name: string, category?: string): CustomComponentOption`

获取自定义组件配置。

**参数**

|参数名|类型|描述|
|---|---|---|
|name|`string`|自定义组件名|
|category|`string`|自定义组件分类，可选|

**返回值**：

* `CustomComponentOption`：自定义组件配置。

**用法**

```javascript
const option = inspector.getComponent('glow', '发光设置')
```

### getComponents

`getComponents(category?: string): Map<string, CustomComponentOption> | undefined`

获取某个分类的所有自定义组件配置，不传参数则取默认分类。

**参数**

|参数名|类型|描述|
|---|---|---|
|category|`string`|自定义组件分类，可选|

**返回值**：

* `Map<string, CustomComponentOption> | undefined`：分类的全部自定义组件配置。

**用法**

```javascript
// 获取默认分类的所有自定义组件配置
const componentsMap = inspector.getComponents()
componentsMap.forEach((value, key) => {
  console.log(`组件${key}的配置为:`, value)
})
```

### getCategories

`getCategories(): string[]`

获取所有自定义组件分类。

**返回值**：

* `string[]`：所有自定义组件分类的名称。

### on

`on(eventName: string, callback: function): void`

监听属性面板事件。

**参数**

|参数名|类型|描述|
|---|---|---|
|eventName|`string`|要监听的事件名称\`|
|callback|`function`|事件触发时要执行的回调函数|

**用法**

```javascript
// 监听属性面板变化
inspector.on('change', (data) => {
  // 其他插件获取 inspector 监听
  const { key, path, value, type, config } = data
  console.log(key, path, value, type, config)
  // ......
  // key: 发生变化的属性的键
  // path: 嵌套键的字符串，用点分隔（例如 'tagsGroup.tags'）
  // value: 发生变化的属性的值
  // type: 发生变化的属性的类型
  // config: 属性面板完整的属性值
})

// 监听添加自定义组件
inspector.on('add-component', (name, category) => {
  console.log('组件名: ' + name)
  console.log('组件分类: ' + category)
}

// 监听移除自定义组件
inspector.on('remove-component', (name, category) => {
  console.log('组件名: ' + name)
  console.log('组件分类: ' + category)
}
```

### off

`off(eventName: string, callback: function): void`

取消监听属性面板事件。

**参数**

|参数名|类型|描述|
|---|---|---|
|eventName|`string`|要取消监听的事件名称|
|callback|`function`|要取消的回调函数|

## 配置规则

单个属性配置项包含通用配置和类型私有配置。  
例如，以下示例中，width 为属性的键，name、type、value、menu 为通用配置，min、max 为类型私有配置。

```javascript
width: {
  name: "宽度",
  type: "number",
  value: 200,
  menu: [
    {
      name: 'rename',
      label: '重命名',
      onClick: (item) => { // 参数为当前属性项
        console.log(item.value)
      }
    },
    { type: 'separator' },
    {
      name: 'reset',
      label: '重置',
      onClick: (item) => {}
    }
  ],
  min: 10,
  max: 1000,
}
```

### 通用配置

|字段名|含义|类型|是否必填|备注|
|---|---|---|---|---|
|name|名称|`string`|是||
|type|类型|`string`|是|参考类型概览|
|value|默认值||否||
|showLabel|是否显示标签|`boolean`|否|默认为 `true`|
|disabled|是否禁用状态|`boolean`|否|默认为 `false`|
|tooltip|提示信息|`string`|否||
|menu|自定义菜单配置|`array`|否||

### 关联配置项

属性面板提供了 `linkage` 字段，一个配置项的显示条件可与其他多个配置项的值所关联。

`linkage` **字段配置规则：**

* linkage 为一个二维数组，子数组之间表示“或”的关系。
* 子数组的成员是配置对象，每个配置对象表示一个关联条件，它们之间表示“且”的关系。
* 当配置对象的关联属性（key）的值匹配关联值（value）时，该条件生效。
* 当所有关联条件都生效时，该配置项会显示。
* 当没有“或”的关系，即二维数组的成员只有一个数组时，可以简写为一维数组。

**示例：**

```javascript
// vehicle 项 的 value 为 'train' 时，trains 项会显示
// vehicle 项 的 value 为 'airplane' 时，flights 项会显示
{
  vehicle: {
    name: '交通工具',
    type: 'select',
    options: [
      {
        value: 'train',
        label: '火车'
      },
      {
        value: 'airplane',
        label: '飞机'
      }
    ],
    value: 'train'
  },
  trains: {
    name: '车次',
    type: 'text',
    linkage: [
      [
        {
          key: 'vehicle',
          value: 'train'
        }
      ]
    ],
    value: '...'

  },
  flights: {
    name: '航班',
    type: 'text',
    linkage: [
      [
        {
          key: 'vehicle',
          value: 'airplane'
        }
      ]
    ],
    value: '...'
  }
}
```

```javascript
// 关联嵌套属性：
// 当 control 的子项 showLight 的 value 为 true 时，light 项会显示
{
  control: {
    name: '控制组',
    type: 'group',
    children: {
      showLight: {
        name: '开启灯光',
        type: 'switch',
        value: false // 此时 light 项不显示
      }
    }
  },
  light: {
    name: '灯光颜色',
    type: 'color',
    value: '#FFFF00',
    linkage: [
      [
        {
          key: 'control.showLight',
          value: true
        }
      ]
    ]
  }
}
```

```javascript
   	value: [0, 0, 0],
   	conversions: [{
   		name: '极坐标', // 转换坐标系名称
   		description: '距离(r)、方位角(θ)、仰角(φ)', // 描述，可选
   		getValue: ([x, y, z]) => {
           const r = Math.sqrt(x * x + y * y + z * z)
           const theta = Math.atan2(y, x) * (180 / Math.PI)
           const phi = Math.acos(z / r) * (180 / Math.PI)
           return [r, theta, phi]
		}, // 将原始坐标系的值转换为目标坐标系的值
		setValue: ([r, theta, phi]) => {
           const thetaRad = theta * (Math.PI / 180)
           const phiRad = phi * (Math.PI / 180)
           const x = r * Math.sin(phiRad) * Math.cos(thetaRad)
           const y = r * Math.sin(phiRad) * Math.sin(thetaRad)
           const z = r * Math.cos(phiRad)
           return [x, y, z]
		} // 将目标坐标系的值转换回原始坐标系的值
	}],
	onUnitChange: (unitName) => {}
}
```
