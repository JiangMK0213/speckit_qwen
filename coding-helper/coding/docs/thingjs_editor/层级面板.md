层级面板用于展示场景层级结构，并支持搜索和自定义交互。

![](https://pan.uino.cn/files/image/2024/9/98d1e735a5633306e264022668e9345e.png)

## 使用层级面板

```javascript
const hierarchy = editor.usePanel('Hierarchy')
```

## 方法

### append

为 Tree 中的一个节点追加一个子节点

**参数**

|参数名|类型|描述|
|---|---|---|
|data|`object`|要追加的 tree 数据|
|parentNode|`string` 或 `object`|父节点的 id 或 data 或 node (不传时往根节点添加)|

**用法**

```javascript
// 定义要添加的 data
const data = {
  id: 'node-id-1',
  iconType: 'icon' // 前面的图标的类型 (icon 或 imageUrl)
  icon: 'icon-name', // 前面的图标的名称 (element-plus 的 icon 名称)
  label: '节点-1',
  children: [
    {
      id: 'node-id-1-1',
      label: '节点-1-1',
      iconType: 'imageUrl',
      icon: '图片路径'
    },
    {
      id: 'node-id-1-2',
      label: '节点-1-2'
    }
  ]
}
// parentNode 不传时往根节点添加
hierarchy.append(data, parentNode)
```

|---|---|---|

**用法**


**用法**

```javascript
hierarchy.insertBefore(data, refNode)
```

|refNode|`string` 或 `object`|参考节点的 data, key 或 node|

```javascript
hierarchy.insertAfter(data, refNode)
```

### setCurrentNode

设置节点为选中状态

**参数**

|参数名|类型|描述|
|---|---|---|

**用法**

```javascript
hierarchy.setCurrentNode(id, true)
```

### setButtons

设置节点的按钮

**参数**

|参数名|类型|描述|
|---|---|---|
|parentNode|`string` 或 `object`|要添加按钮的根节点 id 或 data 或 node|
|buttons|`array`|按钮数组|

buttons 数组中的对象属性:

|属性|类型|描述|
|---|---|---|
|name|`string`|名称（element-plus 的 icon 名称），可选参数|
|className|`string`|按钮的唯一类名，可选参数|
|toggleName|`string`|切换图标的名称（element-plus 的 icon 名称，与 name 搭配使用，可选参数|
|img|`string`|图片路径（当 button 对象中存在 name 属性，将会显示 name 对应的图标，如果没有 name 属性则显示 img），可选参数|
|toggleImg|`string`|切换的图片路径，与 img 搭配使用 ，可选参数|
|onClick|`function`|单击按钮时的回调函数|
|filter|`function`|过滤需要设置按钮的节点|

**用法**

```javascript
hierarchy.setButtons(id, [
  {
    name: 'element-plus 的 icon 名称',
    toggleName: '点击切换的 icon 名称',
    onClick: (data) => {
      console.log(data)
    },
    filter: (data) => {
      // 过滤需要设置按钮的节点
      return data.id === 'test'
    },
    initToggle: (data) => {
      // 根据 data 的某些属性决定初始是否切换
      return data.someCondition
    }
  },
  {
    img: '图片路径',
    toggleImg: '点击切换的图片路径',
    onClick: (data) => {
      console.log(data)
    }
  }
])
```

### setContextMenu

设置 tree 的右键菜单

**参数**

|参数名|类型|描述|
|---|---|---|
|parentNode|`string` 或 `object`|要添加菜单的 tree 的根节点 node 或 id 或 data|
|contextMenuOptions|`array`|菜单项数组|

contextMenuOptions 数组中的对象属性:

|属性|类型|描述|
|---|---|---|
|name|`string`|名称，可选，考虑未来国际化使用|
|label|`string`|显示的文本，必填|
|onClick|`function`|单击菜单项时的回调函数|
|filter|`function`|根据选中节点过滤菜单项|

**用法**

```javascript
// 设置tree的右键菜单
hierarchy.setContextMenu(id, [
  {
    name: 'addToBlueprint',
    label: '添加到蓝图',
    onClick: (event) => {
      // 单击事件，event 基于原生事件对象封装
      console.log(event.data) // event.data，节点数据
    },
    filter: (selectedNodesData) => {
      // 设置过滤条件，selectedNodesData 为当前选中节点数据的数组
      return selectedNodesData.length === 1
    }
  },
  {
    name: 'delete',
    label: '删除',
    onClick: (event) => {
      console.log(event.data)
    }
  }
])
```

### setFilterNodeMethod

设置过滤节点的方法

**参数**

|参数名|类型|描述|
|---|---|---|
|parentNode|`string` 或 `object`|tree 的根节点 node 或 id 或 data|
|filterNodeMethod|`function`|过滤节点的回调函数|

**用法**

```javascript
hierarchy.setFilterNodeMethod(rootId, (value, data, param) => {
  // value 为搜索框的值
  // data 为节点的数据
  // param 为额外参数 {isCaseMatch: false, isExactMatch: false, isRegMatch: false}, 分别对应搜索框的大小写匹配、精确匹配、正则匹配按钮状态

  return data.id === 'test'
})
```

### allowDrop

设置是否允许拖拽

**参数**

|参数名|类型|描述|
|---|---|---|
|parentNode|`string` 或 `object`|tree 的根节点 node 或 id 或 data|
|allowDropFun|`function`|判断节点是否允许释放回调|

**用法**

```javascript
hierarchy.allowDrop(rootId, (draggingNode, dropNode, type) => {
  // draggingNode 为拖拽的节点
  // dropNode 为释放的节点
  // type 为释放的类型

  return draggingNode.data.id === 'test'
})
```

### setControlButtons

设置控制按钮

**参数**

|参数名|类型|描述|
|---|---|---|
|buttons|`array`|控制按钮数组|

**用法**

```javascript
const buttons = [{
    name: 'Add',
    visible: true,
    disabled: false,
    onClick: (name) => { console.log('Clicked' + name) }
}]

hierarchy.setControlButtons(buttons)
```

### getControlButtons

获取控制按钮

**用法**

```javascript
const controlButtons = hierarchy.getControlButtons()

controlButtons[0].disabled = true
```

### on

监听 tree 节点相关事件

**参数**

|参数名|类型|描述|
|---|---|---|
|eventName|`string`|要监听的事件名称，目前只支持 `click` 和 `drag`|
|callback|`function`|事件触发时要执行的回调函数|

**用法**

```javascript
hierarchy.on('click', (data, event) => {
  console.log('nodeClicked', data, event)
})

hierarchy.on('drag', (data, event) => {
  console.log('nodeDrag', data, event)
})
```

### off

移除监听 tree 节点相关事件

**参数**

|参数名|类型|描述|
|---|---|---|
|eventName|`string`|要监听的事件名称，目前只支持 `click` 和 `drag`|
|callback|`function`|事件触发时要执行的回调函数|

**用法**

```javascript
hierarchy.off('click', (data, event) => {
  console.log(data, event)
})

hierarchy.off('drag', (data, event) => {
  console.log(data, event)
})
```
