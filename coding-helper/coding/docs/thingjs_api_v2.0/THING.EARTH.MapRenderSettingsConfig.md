# 类: MapRenderSettingsConfig

**命名空间:** THING.EARTH

**描述:** 空

## 实例属性

### lights

**类型:** `member`

**数据类型:** `Object`

**描述:** 获取灯光配置

---

### cameraEffects

**类型:** `member`

**数据类型:** `Object`

**描述:** 获取发光,拖尾等效果配置

---

### postEffects

**类型:** `member`

**数据类型:** `Object`

**描述:** 获取屏幕级后期效果配置

---

### background

**类型:** `member`

**数据类型:** `Object`

**描述:** 获取背景配置 类型有三种:image,color,cube

---

### environment

**类型:** `member`

**数据类型:** `Object`

**描述:** 获取环境配置 主要包括环境图,漫反射系数,高光系数

---

### visibleLevel

**类型:** `member`

**数据类型:** `Array.<Number>`

**描述:** 获取/设置背景的可见层级

---
