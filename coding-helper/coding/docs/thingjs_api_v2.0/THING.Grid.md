# 类: Grid

**命名空间:** THING

**描述:** 网格对象构造函数

**参数:**

| 名称 | 类型 | 默认值 | 可选 | 描述 |
|------|------|--------|-------------|--------|
| param | `object` | - | 否 | 参数 |
| param.workPlane | `GridWorkPlane` | - | 否 | 网格的平面 |
| param.size | `GridSize` | - | 否 | 网格的大小 |

## 实例属性

### length

**类型:** `member`

**数据类型:** `number`

**描述:** 网格的长度

---

### width

**类型:** `member`

**数据类型:** `number`

**描述:** 网格的宽度

---

### density

**类型:** `member`

**数据类型:** `number`

**描述:** 网格的密度，每个网格的长度(数字必须大于0)

---
