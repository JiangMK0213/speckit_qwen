# coding_use



## 工程描述
>- 用于使用ThingJS API和ThingJS Editor开发的参考文档。

## 工程结构
### 工程目录
```sh
thing-coding-helper
├── docs
│ ├── docs/thingjs_api_v2.0
│ ├── docs/thingjs_editor
├── examples
│ ├── examples/thingjs_api_v2.0
│ ├── examples/thingjs_editor
├── README.md
```

### docs
- 用于存放参考文档。
> docs/thingjs_api_v2.0 存放ThingJS API v2.0的API参考文档。
> docs/thingjs_editor 存放ThingJS Editor的API参考文档。

### examples
- 用于存放示例代码。
> examples/thingjs_api_v2.0 存放ThingJS API v2.0的示例代码。
> examples/thingjs_editor 存放ThingJS Editor的示例代码。
