编辑器支持为场景面板添加工具栏，并提供了一套内置图标库，方便开发者定制场景面板中的交互。

![](https://pan.uino.cn/files/image/2024/9/35237a51404917e4e6a3b9080d4733f7.png)

## 使用工具栏

### 1\. 获取场景面板的 Page 并添加工具栏

```javascript
const scenePanel = editor.getPanel('Scene') // 获取场景面板
const page = scenePanel.getPage('div3d') // 通过 ID 获取页面
const toolbar = page.addToolbar() // 给页面添加工具栏
```

### 2\. 获取和删除工具栏

```javascript
const toolbar = page.getToolbar() // 获取工具栏实例
page.removeToolbar() // 删除工具栏
```

### 3\. 隐藏和显示工具栏

```javascript
toolbar.setVisible(true) // 显示工具栏
toolbar.setVisible(false) // 隐藏工具栏
```

## 工具栏项目操作

### 1\. 添加项目

使用 `addItems` 方法为工具栏添加项目，支持按钮、分割线、下拉菜单等类型。

```javascript
toolbar.addItems([
  {
    id: 'unique-item-id', // 项目唯一标识
    type: 'button', // 项目类型，默认为 button
    icon: 'fa-solid fa-home', // 图标名 (FontAwesome)
    tooltip: 'Home', // 鼠标悬停时显示的提示信息
    onClick: (item) => {
      // 点击项目时触发的回调函数
      toolbar.clearActiveItems() // 清除所有激活状态的项目
      toolbar.setItemActive(item.id, true) // 激活当前项目
    }
  },
  {
    type: 'separator' // 添加分割线
  },
  {
    id: 'dropdown-inspector-id',
    type: 'dropdown-inspector', // 下拉属性面板
    icon: 'fa-solid fa-home', // 图标
    onChange(key, value) {
      console.log(`属性 ${key} 的值变为 ${value}`)
    },
    options: {
      name: {
        name: '名称',
        type: 'text',
        value: '设备x'
      }
    }
  }
])
```

### 2\. 激活和取消激活项目

```javascript
toolbar.setItemActive('unique-item-id', true) // 激活项目
toolbar.setItemActive('unique-item-id', false) // 取消激活项目
toolbar.clearActiveItems() // 清除所有激活项目
```

### 3\. 启用和禁用项目

```javascript
toolbar.setItemEnabled('unique-item-id', true) // 启用项目
toolbar.setItemEnabled('unique-item-id', false) // 禁用项目
```

### 4\. 获取和更新项目

```javascript
const item = toolbar.getItem('unique-item-id') // 获取项目
console.log(item)

toolbar.updateItem('unique-item-id', {
  icon: 'fa-solid fa-cog',
  tooltip: 'Settings'
}) // 更新项目信息
```

### 5\. 移除项目

```javascript
toolbar.removeItem('unique-item-id') // 移除项目
```

## 支持的项目类型

|**类型**|**描述**|
|---|---|
|`button`|按钮类型|
|`separator`|分割线|
|`dropdown`|下拉菜单|
|`select`|下拉选择|
|`dropdown-inspector`|下拉属性面板|
