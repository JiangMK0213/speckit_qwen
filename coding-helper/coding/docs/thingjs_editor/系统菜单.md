本文档介绍如何在系统内置的主菜单中添加、操作和管理菜单项。开发者可以将自己的菜单项添加到系统的主菜单中，或自定义菜单项的结构和行为。系统默认包含以下菜单：文件(File)、编辑(Edit)、开发(Development)、面板(Panels) 和 帮助(Help)。

## 定义菜单项

开发者可以通过定义对象来创建菜单项，并配置其子菜单和行为。

```javascript
// 定义一个菜单项及其子菜单
const menuItem = {
  id: 'Package',  // 菜单项的唯一 ID
  label: '包管理', // 菜单项的显示文本
  submenu: [      // 子菜单项
    {
      id: 'Import',
      label: '导入',
      onClick: () => {
        console.log('Import...')
      }
    },
    {
      id: 'Export',
      label: '导出',
      onClick: () => {
        console.log('Export...')
      }
    },
    {
      type: 'separator' // 分割线，用于视觉分隔子菜单项
    },
    {
      id: 'About',
      label: '关于',
      submenu: [
        {
          id: 'About Uino',
          label: '关于 Uino',
          onClick: () => {}
        },
        {
          id: 'About ThingJS',
          label: '关于 ThingJS',
          onClick: () => {}
        }
      ]
    }
  ]
}
```

## 添加菜单

### 添加菜单项到主菜单

```javascript
editor.menu.addItem(menuItem)  // 将自定义菜单项添加到主菜单
```

### 添加菜单项到“文件”菜单

```javascript
<code>const fileMenu = editor.menu.getItem('File')  // 根据 ID 获取“文件”菜单
const packageMenu = fileMenu.addItem(menuItem) // 添加到“文件”菜单
console.log(packageMenu)  // 打印添加后的菜单项
```

### 在指定菜单项之前添加

```javascript
editor.menu.addItemBefore(menuItem, 'refId')  // 在 refId 前添加菜单项
```

### 在指定菜单项之后添加

```javascript
editor.menu.addItemAfter(menuItem, 'refId')  // 在 refId 后添加菜单项
```

### 添加到指定菜单的末尾

```javascript
editor.menu.addItemEndOf(menuItem, 'refId')  // 在 refId 所在菜单末尾添加
```

## 菜单项管理

### 获取所有系统菜单

```javascript
<code><code><code>const menus = editor.menu.getItems()  // 获取所有系统菜单
console.log(menus)
```

### 根据 ID 删除菜单项

```javascript
editor.menu.removeItem('Export')  // 删除 ID 为 'Export' 的菜单项
```

### 启用菜单项

```javascript
editor.menu.enableItem('Export')  // 启用 ID 为 'Export' 的菜单项
```

### 禁用菜单项

```javascript
editor.menu.disableItem('Export')  // 禁用 ID 为 'Export' 的菜单项
```

### 判断菜单项是否启用

```javascript
<code><code><code>const exportMenu = editor.menu.getItem('Export')  // 获取 'Export' 菜单项
console.log(exportMenu.isEnabled())  // 打印是否启用状态 (true/false)
```
