资源面板组件用于管理和展示资源，包含资源页签、项目列表和右键菜单。开发者可以通过资源面板轻松添加、删除资源页签，并对资源进行操作，如重命名、复制、导入等。

![](https://pan.uino.cn/files/image/2024/10/a3a15e21c984e20f210f038a41016c30.png)

## 使用资源面板

```javascript
// 使用资源面板（添加到编辑器中并获取）
const resource = editor.usePanel('Resource')
```

## 资源面板方法

### addTab(tabOptions)

向资源面板中添加一个新的页签。该方法为异步方法，需进行 `await` 操作。

**参数**

|参数名|类型|描述|
|---|---|---|
|tabOptions|`object`|页签配置对象|

tabOptions 配置项：

|属性|类型|描述|
|---|---|---|
|name|`string`|页签的名称，必填，唯一|
|label|`string`|显示的文本，可选，不填时取 name 的值|
|dirName|`string`|资源目录名称|
|accept|`string`|导入资源接受类型, 默认为".zip"|
|ignoreNames|`array`|忽略的文件夹名, 仅在 isFolder 为 true 的情况下生效|
|autoImport|`boolean`|控制导入资源的自动化程度|
|enableGroup|`boolean`|控制是否允许分组操作，如果设置为`false`，顶部工具栏中将不显示`+`按钮|
|allowCreate|`boolean`|控制是否允许新建资源，如果为`false`，顶部工具栏以及分组下拉菜单中将不显示 `新建资源`选项|
|allowRename|`boolean`|控制是否允许重命名资源，如果为`false`，资源右键菜单中将不显示`重命名`选项|
|allowCopy|`boolean`|控制是否允许复制资源，如果为`false`，资源右键菜单中将不显示`复制`选项|
|allowRemove|`boolean`|控制是否允许删除资源，如果为`false`，资源右键菜单中将不显示`删除`选项|
|allowOpenFolder|`boolean`|控制是否允许打开文件夹，如果为`false`，资源右键菜单中将不显示`打开文件夹`选项|
|itemSize|`string`|`'large'` 1 个/行，`'small'` 3 个/行，默认为 `'small'`|
|order|`number`|页签在资源面板中的显示顺序|
|onClickItem|`function`|单击资源时的回调函数|
|onDblClickItem|`function`|双击资源时的回调函数|
|onDragItem|`function`|拖动资源时的回调函数|
|onActiveItem|`function`|资源设置 active 状态时的回调函数|
|onDeactiveItem|`function`|资源取消 active 状态时的回调函数|
|onCreateItem|`function`|添加资源时的回调函数|
|onBeforeRenameItem|`function`|重命名资源前的回调函数|
|onRenameItem|`function`|重命名资源后的回调函数|
|onBeforeRemoveItem|`function`|删除资源前的回调函数|
|onRemoveItem|`function`|删除资源后的回调函数|
|onCopyItem|`function`|复制资源后的回调函数|
|onImportItem|`function`|导入资源后的回调函数|

**用法**

```javascript
// 添加资源页签
const tab = await resource.addTab({
  name: 'test',
  label: '测试',
  itemSize: 'small',
  order: 5,
  dirName: 'test',
  accept: '.zip, .bundle, .tjs',
  ignoreNames: ['images'],
  autoImport: false,
  allowCreate: true,
  allowRename: true,
  allowCopy: true,
  allowRemove: true,
  onActiveItem: (ev) => {
    console.log('onActiveItem', ev.item)
  },
  onDeactiveItem: (ev) => {
    console.log('onDeactiveItem', ev.item)
  },
  onClickItem: (event) => {
    console.log(event.item)
  },
  onDblClickItem: (event, ev) => {
    console.log('onDblClickItem', event.item)
  },
  onCreateItem: (group) => {
    console.log('createItem', group)
  },
  onDragItem: (event) => {
    console.log(event.item, 'drag')
  },
  onBeforeRenameItem: (item) => {
  	if (item.name === 'test') {
      return false
    }
    return true
  },
  onRenameItem: (item) => {
    console.log('onRenameItem', item)
  },
  onBeforeRemoveItem: (item) => {
    if (item.name === 'test') {
      return false
    }
    return true
  },
  onRemoveItem: (item) => {
    console.log('onRemoveItem', item)
  },
  onCopyItem: (item) => {
    console.log('onCopyItem', item)
  },
  onImportItem: (item, group) => {
    console.log('importItem', item, group)
  }
})
```

### removeTab(tabName)

删除一个指定的资源页签。

**参数**

|参数名|类型|描述|
|---|---|---|
|tabName|`string`|要删除的页签的名称|

**用法**

```javascript
// 删除资源页签
resource.removeTab('MyType')
```

### getActiveTab()

获取当前激活的页签。

**返回值**

* `object`：当前激活的页签实例。

**用法**

```javascript
// 获取当前激活的 Tab
const tab = resource.getActiveTab()
console.log(tab.name)
```

## 资源页签方法

### getItem(itemName)

获取指定的资源。

**参数**

|参数名|类型|描述|
|---|---|---|
|itemName|`string`|资源名称|

**用法**

```javascript
const resourceItem = tab.getItem('prefab1')
```

### updateItem(item, updatedValues)

更新指定的资源。

**参数**

|参数名|类型|描述|
|---|---|---|
|item|`object`|资源对象|
|updatedValues|`object`|更新的值|

**用法**

```javascript
tab.updateItem(itemObj, {
  preview: 'preview1.png'
})
```

### setContextMenu(contextMenuOptions)

设置页签的右键菜单。

**参数**

|参数名|类型|描述|
|---|---|---|
|contextMenuOptions|`array`|菜单项数组|

contextMenuOptions 数组中的对象属性:

|属性|类型|描述|
|---|---|---|
|name|`string`|名称，可选，考虑未来国际化使用|
|label|`string`|显示的文本，必填|
|onClick|`function`|单击菜单项时的回调函数|

**用法**

```javascript
// 设置页签的右键菜单
tab.setContextMenu([
  {
    name: 'addToScene',
    label: '添加到场景',
    onClick: (event) => {
      // 单击事件，event 基于原生事件对象封装
      console.log(event.item) // event.item，项目数据 { name: xxx, url: xxx, ... }
    }
  }
])
```

### setActiveItem(item, isActive, isScrollToItem)

设置资源的 active 状态。

**参数**

|参数名|类型|描述|
|---|---|---|
|item|`object`|资源对象|
|isActive|`boolean`|是否激活|
|isScrollToItem|`boolean`|是否定位到资源, 默认值为`true`|

**用法**

```javascript
tab.setActiveItem(itemObj, true, false)
tab.setActiveItem(itemObj, false)
```

### setCurrentItem(item, isCurrent, isScrollToItem)

设置资源的 current 状态。

**参数**

|参数名|类型|描述|
|---|---|---|
|item|`object`|资源对象|
|isCurrent|`boolean`|是否当前|
|isScrollToItem|`boolean`|是否定位到资源, 默认值为`true`|

**用法**

```javascript
tab.setCurrentItem(itemObj, true, false)
tab.setCurrentItem(itemObj, false)
```
