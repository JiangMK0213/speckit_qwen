# 类: RoomLevelControl

**命名空间:** THING.CAMPUS

**继承:** [THING.DefaultLevelControl](./THING.DefaultLevelControl.md)

**描述:** 空

## 实例属性

### isObjectTransparent

**类型:** `member`

**数据类型:** `boolean`

**描述:** 开启/关闭 物体半透明。(默认关闭，开启后进入房间时会半透明其他房间下的物体。并且在继续进入子层级时不会主动恢复其他房间的半透明，只有返回父层级才会恢复其他房间的半透明。)
