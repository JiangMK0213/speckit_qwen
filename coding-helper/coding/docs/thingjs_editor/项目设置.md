ThingJS Editor 提供了 `**settings**` 模块，用于存储和读取项目的自定义数据，这些数据可以在多个插件间共享。该功能支持嵌套数据结构，并允许开发者以灵活的方式管理项目状态和配置。

## 使用示例

### 1\. 设置数据

通过 `editor.settings.set` 将数据存储在项目的设置中。`key` 为字符串类型，`value` 为可序列化的 JSON 对象：

```javascript
editor.settings.set('theme', { color: 'blue', fontSize: 14 })
```

### 2\. 获取数据

使用 `editor.settings.get` 根据 `key` 获取对应的设置值：

```javascript
const theme = editor.settings.get('theme')
console.log(theme) // 输出: { color: 'blue', fontSize: 14 }
```

### 3\. 删除数据

通过 `editor.settings.delete` 删除指定 `key` 的数据：

```javascript
editor.settings.delete('theme')
```

### 4\. 嵌套数据支持

项目设置支持通过 `**.**` 分隔符设置和获取嵌套对象的属性：

```javascript
// 设置嵌套属性
editor.settings.set('a.b', 123)

// 获取整个对象
const obj = editor.settings.get('a')
console.log(obj) // 输出: { b: 123 }

// 获取嵌套属性
const value = editor.settings.get('a.b')
console.log(value) // 输出: 123
```
