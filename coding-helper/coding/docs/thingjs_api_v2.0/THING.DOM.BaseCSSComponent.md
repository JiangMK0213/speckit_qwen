# 类: BaseCSSComponent

**命名空间:** THING.DOM

**继承:** [THING.EventCaptureComponent](./THING.EventCaptureComponent.md)

**描述:** 空

## 实例属性

### active

**类型:** `member`

**数据类型:** `boolean`

**描述:** 获取/设置是否激活组件。

---

### enableEventProxy

**类型:** `member`

**数据类型:** `boolean`

**描述:** 获取/设置是否启用事件代理。<br>True 表示可以触发对象的事件，例如 'click', 'mousemove' 等。

---

### visible

**类型:** `member`

**数据类型:** `boolean`

**描述:** 获取/设置可见状态，null 表示自动更新 DOM 元素的可见状态。

---

### zIndex

**类型:** `member`

**数据类型:** `number`

**描述:** 获取/设置 DOM 元素的 Z-Index 值，null 表示自动设置 Z-Index。

---

### interactive

**类型:** `member`

**数据类型:** `boolean`

**描述:** 获取/设置是否可交互。

---

### glassBlur

**类型:** `member`

**数据类型:** `number`

**描述:** 获取/设置玻璃模糊效果值（以像素为单位），默认值为 0。

---

### domElement

**类型:** `member`

**数据类型:** `object`

**描述:** 获取/设置DOM元素。

---
