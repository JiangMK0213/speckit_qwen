本指南介绍了编辑器提供给插件使用的内置事件。这些事件涵盖了编辑器的运行状态、插件的管理以及面板的操作等常见场景。插件可以通过注册事件来响应这些编辑器行为，从而实现扩展功能。

## 事件分类

* 系统状态事件：准备完毕、保存、运行/停止、撤销/恢复
* 插件管理事件：插件安装、卸载、更新、参数改变
* 面板操作事件：激活、聚焦/失焦、改变大小、关闭

## 系统状态事件

### 准备完毕

所有插件安装完成时触发。

```javascript
editor.on('ready', () => {
  console.log('编辑器准备完毕')
})
```

### 保存

编辑器会在保存操作时触发该事件，收集所有插件中的 `save` 事件处理结果，并在完成后触发 `saved` 事件。

```javascript
// 监听保存事件
editor.on('save', () => {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      console.log('保存完成')
      resolve()
    }, 2000) // 模拟异步保存
  })
})

// 监听所有保存完成后的事件
editor.on('saved', () => {
  console.log('所有插件保存完成')
})
```

## 运行/停止

插件可在编辑器运行或停止时执行异步操作，并在操作完成后，编辑器会触发对应的 `playing` 事件。

```javascript
// 监听运行事件
editor.on('play', () => {
  return new Promise((resolve) => {
    setTimeout(() => {
      console.log('运行完成')
      resolve()
    }, 2000)
  })
})

// 所有插件运行完成后触发
editor.on('playing', () => {
  console.log('编辑器已进入运行状态')
})

// 监听停止事件
editor.on('stop', () => {
  console.log('编辑器已停止')
})
```

## 撤销/恢复

支持操作的撤销与恢复。

```javascript
// 监听撤销事件
editor.on('undo', () => {
  console.log('撤销操作')
})

// 监听恢复事件
editor.on('redo', () => {
  console.log('恢复操作')
})
```

## 插件管理事件

### 插件安装/卸载/更新

当插件安装、卸载或更新时触发。

```javascript
// 插件安装完成
editor.on('plugin:install', (name) => {
  console.log(`插件安装完成: ${name}`)
})

// 插件卸载完成
editor.on('plugin:uninstall', (name) => {
  console.log(`插件卸载完成: ${name}`)
})

// 插件更新（热更新）完成
editor.on('plugin:update', (name) => {
  console.log(`插件更新完成: ${name}`)
})
```

### 插件参数改变

当插件参数发生变化时触发。

```javascript
editor.on('plugin:change-params', (name, data) => {
  console.log(`插件参数改变: ${name}`)
  console.log('参数:', data) // { "key": "value", ... }
})
```

## 面板操作事件

### 面板激活

当某个面板被激活时触发。

```javascript
editor.on('panel-active', (id) => {
  console.log(`面板激活: ${id}`)
})
```

### 面板聚焦/失焦

监听面板的聚焦与失焦状态。

```javascript
// 面板聚焦
editor.on('panel-focus', (id) => {
  console.log(`面板聚焦: ${id}`)
})

// 面板失焦
editor.on('panel-blur', (id) => {
  console.log(`面板失焦: ${id}`)
})
```

### 面板改变大小

当面板大小发生变化时触发。

```javascript
editor.on('panel-resize', (id) => {
  console.log(`面板大小改变: ${id}`)
})
```

### 面板关闭

当面板被关闭时触发。

```javascript
editor.on('panel-close', (id) => {
  console.log(`面板关闭: ${id}`)
})
```
