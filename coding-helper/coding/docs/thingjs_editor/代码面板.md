代码面板支持用户编写自己的代码逻辑，通常结合蓝图一起使用。

![](https://pan.uino.cn/files/image/2024/9/d381e1b94dff75cebe270d73372831d9.png)

## 使用代码面板

```javascript
const code = editor.usePanel('Code')
```

## 方法

### setValue

`setValue(value: string): void`

设置代码面板的当前内容。

**参数**

|参数名|类型|描述|
|---|---|---|
|value|`string`|要设置的代码文本|

**用法**

```javascript
code.setValue('Hello, world!')
```

### getValue

`getValue(): string`

获取代码面板的当前内容。

**返回值**

* `string`：代码面板的当前文本内容。

**用法**

```javascript
const currentCode = code.getValue()
console.log(currentCode)
```

### on

`on(eventName: string, callback: function): void`

监听代码面板相关事件。

**参数**

|参数名|类型|描述|
|---|---|---|
|eventName|`string`|要监听的事件名称，目前只支持 `'change'`|
|callback|`function`|事件触发时要执行的回调函数|

**用法**

```javascript
code.on('change', (newValue) => {
  console.log('Code changed:', newValue)
})
```
