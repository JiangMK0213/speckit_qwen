# 类: CSS3DNode

**命名空间:** THING.DOM

**描述:** constructor.Initializes the CSS3DNode with the specified render type.

**参数:**

| 名称 | 类型 | 默认值 | 可选 | 描述 |
|------|------|--------|-------------|--------|
| renderType | `string` | - | 否 | The type of rendering to be applied. |
