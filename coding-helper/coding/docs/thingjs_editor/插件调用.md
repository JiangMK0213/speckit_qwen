编辑器提供了 `**getPlugin**` 接口，通过插件名称获取对应的插件实例，并调用其在入口文件中导出的方法。此方法仅适用于 **当前激活的插件**。

## 插件调用示例

1. 获取插件实例
2. 调用插件方法

### 代码示例

```javascript
// 根据插件名称获取插件实例
const plugin = editor.getPlugin('plugin-name') // 'plugin-name' 为插件 config.json 中的名称

// 调用插件导出的方法
if (plugin && typeof plugin.someMethod === 'function') {
  plugin.someMethod() // 执行 someMethod 方法
}
```

### 插件入口文件示例

在插件的入口文件中，需要导出提供的方法，以便编辑器能通过插件实例调用。

```javascript
// 插件方法示例
function someMethod() {
  console.log('someMethod 被调用')
}

// 定义生命周期钩子
async function onBeforeInstall() {
  console.log('插件安装前执行')
}

async function onInstall() {
  console.log('插件安装时执行')
}

async function onUninstall() {
  console.log('插件卸载时执行')
}

// 导出插件的生命周期钩子和方法
export default {
  onBeforeInstall,
  onInstall,
  onUninstall,
  someMethod // 导出插件方法
}
```
