编辑器支持为场景面板添加 **快速访问栏（Quickbar）**，并提供内置图标库，方便开发者快速创建自定义交互工具。

![](https://pan.uino.cn/files/image/2024/9/6434fe0ad1b7dd859b23fed899920aba.png)

## 使用快速访问栏

### 1\. 添加 Quickbar

```javascript
const scenePanel = editor.getPanel('Scene') // 获取场景面板
const page = scenePanel.getPage('div3d') // 通过 ID 获取页面
const quickbar = page.addQuickbar({
  id: 'quickbar-1', // Quickbar 的唯一标识
  position: 'top' // 位置: 'top', 'bottom', 'left', 'right'
})
```

### 2\. 获取和删除 Quickbar

```javascript
const quickbar = page.getQuickbar('quickbar-1') // 获取 Quickbar
page.removeQuickbar('quickbar-1') // 删除 Quickbar
```

### 3\. 显示和隐藏 Quickbar

```javascript
quickbar.setVisible(true) // 显示 Quickbar
quickbar.setVisible(false) // 隐藏 Quickbar
```

## Quickbar 项目操作

### 1\. 添加项目

使用 `addItems` 方法为 Quickbar 添加项目。支持多种类型项目，如按钮、分割线和下拉菜单。

```javascript
quickbar.addItems([
  {
    id: 'unique-item-id', // 唯一标识
    type: 'button', // 支持类型：button, dropdown-inspector, separator
    icon: 'fa-solid fa-home', // 使用 FontAwesome 图标库
    tooltip: 'Home', // 鼠标悬停提示
    onClick: (item) => {
      quickbar.clearActiveItems() // 清除所有激活状态
      quickbar.setItemActive(item.id, true) // 激活当前项目
    }
  },
  {
    type: 'separator' // 添加分割线
  },
  {
    id: 'dropdown-inspector-id',
    type: 'dropdown-inspector', // 下拉属性面板
    icon: 'fa-solid fa-home', // 图标
    onChange(key, value) {
      console.log(`属性 ${key} 的值变为 ${value}`)
    },
    options: {
      name: {
        name: '名称',
        type: 'text',
        value: '设备x'
      }
    }
  }
])
```

### 2\. 激活和取消激活项目

```javascript
quickbar.setItemActive('unique-item-id', true) // 激活项目
quickbar.setItemActive('unique-item-id', false) // 取消激活项目
quickbar.clearActiveItems() // 清除所有激活项目
```

### 3\. 启用和禁用项目

```javascript
quickbar.setItemEnabled('unique-item-id', true) // 启用项目
quickbar.setItemEnabled('unique-item-id', false) // 禁用项目
```

### 4\. 获取和更新项目

```javascript
const item = quickbar.getItem('unique-item-id') // 获取项目
console.log(item)

quickbar.updateItem('unique-item-id', {
  icon: 'fa-solid fa-cog',
  tooltip: 'Settings'
}) // 更新项目信息
```

### 5\. 移除项目

```javascript
quickbar.removeItem('unique-item-id') // 移除项目
```

## 支持的项目类型

|**类型**|**描述**|
|---|---|
|`button`|按钮类型|
|`separator`|分割线|
|`dropdown`|下拉菜单|
|`select`|下拉选择|
|`dropdown-inspector`|下拉属性面板|
